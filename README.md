# APlusMerchandising
A+ Merchandising specializes in premium medical apparel designed for healthcare professionals, students, doctors, and nurses. All products are crafted with a focus on comfort, durability, hygiene, and professional aesthetics, suitable for daily long-shift usage.

## Website
A demo, dark-themed static website has been added to the `site/` folder. It includes a modern homepage, product previews, a demo client-side cart (localStorage), and a sample contact form. Replace placeholder images and form endpoints (Formspree or backend) before going live. See `site/README.md` for preview and deployment instructions.
